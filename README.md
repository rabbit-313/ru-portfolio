# Ru - ML/AI エンジニアリング

個人事業「Ru」のビジネスホームページです。MLモデルの開発からインフラ構築、技術コンサルティングまで、ML/AI領域のサービスを紹介しています。

## ライブサイト

**[https://rabbit-313.github.io/ru-portfolio/](https://rabbit-313.github.io/ru-portfolio/)**

## 提供サービス

- **MLモデル開発**: 推薦システム、画像認識、超解像など
- **MLインフラ構築**: MLパイプライン、GCPインフラ、MLOps導入
- **技術コンサルティング**: ML/AI導入検討、技術選定、PoC

## 技術スタック（サイト）

- HTML5, CSS3, JavaScript
- Font Awesome
- GitHub Pages + GitHub Actions

## ローカル開発

```bash
git clone https://github.com/rabbit-313/ru-portfolio.git
cd ru-portfolio
python -m http.server 8000
```
